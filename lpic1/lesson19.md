# vim
	`k` ----> up
	`j` ----> down
	`w` ----> jump to next word(first letter of it)
	`e` ----> like `w` but jump for the last letter of the word
	`b` ----> like w but for previous word
	`ctrl+f` ---> each time pressing `ctrl` and each tap of `f` ---> scroll 1         page forward
	`ctrl+b` ---> like `ctrl+f` but backward
	`double g` -----> to  the beginning of the file
	`shift+g` ----> to the end of the file
	- ***tip*** ---> `number`+`shortcut` ----> runs that short for `number` times     ===> `3w` ---> goes to 3 next words
	- `/` ===> searching
	- `n` ---> go next
	- `N` ---> go previous
	- `ZZ` ----> save and exit
	- `:e!` ----> reload the file
	- `:!command` ---> run shell commands
	- `:w newfile` ----> write to the newfile
